To set the channel banner image, please follow these guidelines:


  • Supported formats: JPEG, PNG
  • Recommended dimensions: 970 x 250 pixels
  • Maximum file size: 5MB or less

Please ensure that your image meets these requirements for the best display quality.

To set the profile picture for your channel, please follow these guidelines:


  • Supported formats: JPEG, PNG, GIF(no animation)
  • Recommended dimensions: 200 x 200 pixels
  • Maximum file size: 3MB

Please ensure that your image meets these requirements for the best display quality.
